🔍 대학별 실제 입시 분석

출처: 각 대학 입학처·에듀진·나무아카데미·진학사 (2025~2026학년도 기준)
🔵 안양대 미디어콘텐츠 — 6등급대 도전 가능
전형: 아리학생부종합Ⅰ (1단계 서류100% → 2단계 서류70%+면접30%)
면접: 블라인드 면접 (교복·이름·학교명 노출 금지), 면접관 2인 × 10분, 생기부 기반 질문
수능최저: 없음 ✅
합격 내신: 미디어콘텐츠 공식 미공개. 인문계 유사학과 70%컷 약 4~5등급대
6등급 전략: 블라인드 면접이라 학교·성적 배경이 드러나지 않음. 연극부→콘텐츠 기획 스토리로 면접에서 역전 가능
⚠️ 실제 70%컷은 안양대 입학처 PDF 직접 확인 필요
🟡 한신대 미디어영상광고 — 경쟁률 20:1 이상, 면접 필수
전형: 참인재전형 (1단계 서류100% → 2단계 서류70%+면접30%)
면접: 개인 면접 (블라인드 여부 미공식 확인)
수능최저: 없음 ✅
합격 내신: 인문계열 70%컷 약 4등급대 / 경쟁률 21.57:1 (2025학년도)
6등급 전략: 경쟁이 매우 치열. 면접에서 압도적인 차별화 없으면 합격 어려움
⚠️ 적정이 아닌 소신으로 분류 검토 필요
🔵 강남대 미디어커뮤니케이션 — 학종 전략 추천
전형A: 지역균형 — 교과70%+면접30% → 6.23등급엔 불리
전형B: 서류면접(학종) — 1단계 서류100% → 2단계 서류70%+면접30% → ✅ 이쪽이 유리
면접: 면접관 3인 × 1인 구술면접, 교과 지식 없음, 생기부 기반 인성면접
수능최저: 없음 ✅
합격 내신: 인문계열 70%컷 약 4~5등급대
6등급 전략: 학종 전형으로 지원, 생기부 스토리 + 3인 면접에서 리더십·기획력 어필
⭐ 명지전문대 연극영상 — 실기+연극부 직접 연결, 최우선 준비
전형: 수시 1차 (실기고사+면접)
실기: 연극영상학과 실기 있음 — 수빈이 SAN학원 실기 준비 직접 연결 ✅
면접: 실기 이후 면접 진행
수능최저: 확인 필요
합격 내신: 약 4.5등급 (2025학년도 평균), 경쟁률 50.5:1
6등급 전략: 실기·면접 비중이 높아 내신 6등급도 충분히 도전 가능. 연극부 부기장 경력이 가장 직접적으로 연결되는 학과. 실기 완성도가 당락 결정
⚠️ 경쟁률 50:1 — 실기 준비에 충분한 투자 필수
🟢 신구대 미디어콘텐츠 — 안정권, 우수 2학기만 반영
전형: 일반전형 (학생부 교과 중심)
내신 반영: 고1~고3-1 중 성적 우수 2개 학기만 반영 → 특정 학기가 좋으면 유리
면접: 없음 ✅ (항공·e스포츠 등 일부 학과만 실시)
수능최저: 없음 ✅
합격 내신: 공식 입결 공개 중 (신구대 입학처 홈페이지 확인 필요)
6등급 전략: 1학년 2학기·2학년 2학기처럼 상대적으로 나은 학기 2개를 골라 유리하게 적용. 면접 없어서 서류만 준비하면 됨
🟢 인덕대 방송영상미디어 — 수시 2차 안전권
전형: 일반전형 (학생부 교과 위주)
면접: 없음 (일반전형 기준)
수능최저: 없음 ✅
합격 내신: 수시 1차 평균 약 4.0~4.1등급, 최저 약 4.6~4.8등급 (2025학년도). 수시 2차에서 합격선 완화
시설: UHD 스튜디오·영상 제작 인프라 우수, 서울 노원
6등급 전략: 수시 1차보다 2차에서 합격 가능성 높아짐. 편입 실적 우수 (안정적인 백업)
